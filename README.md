# Domingo Cars Luxury Rent — Frontend

React 18 + Vite 4 frontend for the Domingo Cars Luxury Rent car rental platform.

## Stack
- React 18, Vite 4, Tailwind CSS v3, React Router v6
- API: https://api.domingocars.ma

## Deploy
Pushes to `main` automatically build and deploy to Hostinger via GitHub Actions.
